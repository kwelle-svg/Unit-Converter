# UNIT-CONVERTER
Unit-Converter that I maked for roadmap.sh project: https://roadmap.sh/projects/unit-converter

![How it's looks](/images/screenshot.png)

**Tech used:** FastAPI, pytest, html, css

**Installation**
``` bash
git clone https://github.com/kwelle-svg/Unit-Converter
python -m venv .venv
source .venv/bin/activate
pip install -r req.txt
```
**Tests**
``` bash
pytest -vv
```
**Use it**
``` bash
uvicorn app.main:app
```
Than open website and use it :)
